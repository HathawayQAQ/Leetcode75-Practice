## Problem Description ##
> Given a string `s` and an integer `k`, return the maximum number of vowel letters in any substring of `s` with length `k`.
>
> Vowel letters in English are `'a'`, `'e'`, `'i'`, `'o'`, and `'u'`.

## First Solution ##
```python
class Solution:
    def maxVowels(self, s: str, k: int) -> int:
        vowels = set("aeiou")
        count = 0

        # initial window [0, k)
        for i in range(k):
            if s[i] in vowels:
                count += 1

        best = count

        # sliding window: [i - k + 1, i]
        for i in range(k, len(s)):
            if s[i - k] in vowels: # left
                count -= 1
            if s[i] in vowels: # right
                count += 1
            if count > best:
                best = count

        return best
```

once move one to right:

- left out: if vowel, count -= 1
- right in: if vowel, count += 1
