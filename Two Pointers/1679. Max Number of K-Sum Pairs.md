## Problem Description ##
> You are given an integer array `nums` and an integer `k`.
>
> In one operation, you can pick two numbers from the array whose sum equals `k` and remove them from the array.
>
> Return the maximum number of operations you can perform.

## First Solution - Hash Count ##
```python
class Solution:
    def maxOperations(self, nums: List[int], k: int) -> int:
        count = {}
        ans = 0

        for x in nums:
            need = k - x
            if count.get(need, 0) > 0:
                ans += 1
                count[need] -= 1
            else:
                count[x] = count.get(x, 0) + 1

        return ans
```

- count.get(need, 0):

  - if key exist, return count[key]

  - if doesn't exist, return 0

  - same as:

    ```python
    if need in count:
      value = count[need]
    else:
      value = 0
    ```

### Improved solution - Two Pointers after Sorting

```python
class Solution:
    def maxOperations(self, nums: List[int], k: int) -> int:
        nums.sort()
        l = 0
        r = len(nums) - 1
        ans = 0

        while l < r:
            s = nums[l] + nums[r]
            if s == k:
                ans += 1
                l += 1
                r -= 1
            elif s < k:
                l += 1
            else:
                r -= 1

        return ans
```
